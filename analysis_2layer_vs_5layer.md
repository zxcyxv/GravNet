## 5. W_down + dw_conv 성장 패턴 (ConvSwiGLU 경로)

### 2-Layer: b0_W_down 시계열
```
step    0: 0.126 → step 2000: 0.266 → step 4950: 0.187
step    0: b1_W_down 0.166 → step 2000: 0.354 → step 4950: 0.383
```
2-Layer에서 b1_W_down이 **단조 증가**하며 전체 학습을 지배. 최종적으로 b1_W_down(0.383)이 lm_head(0.498) 다음으로 가장 큰 gradient.

### 5-Layer: b0_W_down 시계열
```
step    0: 0.062 → step 2000: 0.097 → step 4950: 0.026
step    0: b1_W_down 0.077 → step 2000: 0.100 → step 4950: 0.043
```
5-Layer에서는 blocks 0-1의 W_down이 **감소 추세**. 깊은 블록(2-4)으로 학습 부담이 이동.

- 5-Layer: lm_head 비중이 **15%**로 낮아지고, gradient가 전 블록에 균등 분배 → **건강한 학습 역학**

## 10. 결론

### 2-Layer의 한계
- 전체 학습 용량이 blocks 0-1에 집중
- 학습 후반 lm_head가 gnorm의 68%를 독점 → **내부 블록 포화**
- pos_emb gradient 소멸 → 위치 정보 활용 부족

### 5-Layer의 특이점 (Singularity)
- blocks 2-4가 전체 gradient의 50-75%를 흡수하며 **학습의 주력**으로 부상
- lm_head 비중이 15%로 균등화 → 모든 블록이 활발히 학습
- pos_emb gradient 5x 유지 → 위치 인코딩의 적극적 활용
- carry reset 스파이크 진폭 증가 → 더 강한 학습 역학
- m_norm/C_norm 성장 → 더 공격적인 mean shift 클러스터링

### 왜 "특이점"인가?
2-Layer → 5-Layer로의 전환은 단순한 용량 증가가 아님. **학습 역학의 질적 전환**:
1. 얕은 블록(0-1)이 feature extraction에서 fine-tuning 역할로 전환
2. 깊은 블록(2-4)이 주요 표현 학습을 담당
3. gradient 분배가 lm_head-중심에서 전 네트워크 균등으로 변화
4. Post-Addition Norm이 5개 블록 ×16 loops = 80 residual add에서도 분산을 안정적으로 유지
